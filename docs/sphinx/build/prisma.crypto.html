verify_local_event(ev)¶

Verifies a signed Event.

Parameters:ev –

A named event tuple with the type, Event_(‘d p t c s’) where;

  • d: Data/payload!
  • p: event-hash of 2 parents (latest events) of event
  • t: time of event creation
  • c: identifying key of first parent
  • s: digital sign of event by first parent (using his secret key)
Returns:True: Successfully verified event.
Returns:False: Unsuccessfully verified event.
Return type:bool
